# NLU Portal Optimization
# Project Overview
Our team project focused on improving the NLU student portal by addressing major usability and performance issues experienced by students. Through a structured survey, we analyzed authentication barriers, navigation challenges, and system-level friction points that impact students’ daily academic workflows. The objective was to transform student-reported problems into actionable design improvements.
# Problem being solved
NLU students face significant difficulties when using the NLU portal. Based on survey data collected from students in our CSS class, the most common issues include:
-Accessing schedules
-Administrative services
-Slow loading times
-Difficult navigation
-Certain pages failing to load
# The core problem
Students experience friction during login and navigation workflows, reducing efficiency and increasing frustration when interacting with university digital systems.
# System Vision
Our vision was to create a more seamless, secure, and intuitive digital access experience for undergraduate students.
The improved system should:
-Improve navigation clarity
-Enhance overall digital reliability
-Reduced loading times
-Fully functional pages
-Faster access to academic resources
# System requirements
AI assistant integration to help students find resources
Backend optimization to reduce loading time
Error detection and fixes for non-loading pages
